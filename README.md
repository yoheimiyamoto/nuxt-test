# nuxt-test
nuxtをいろいろ勉強してみる


# 環境構築
```
// cliインストール
npm i -g @vue/cli @vue/cli-init
```

# プロジェクト作成
// プロジェクト作成
vue init nuxt-community/starter-template {アプリ名}

// アプリのディレクトリに移動
cd {アプリ名}

// インストール
npm install

// サーバー起動
npm run dev

// サイトにアクセス
http://localhost:3000/
```
